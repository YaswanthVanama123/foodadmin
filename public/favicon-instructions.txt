FAVICON INSTRUCTIONS FOR ADMIN PANEL
=====================================

This file serves as a placeholder and guide for creating the admin panel favicon.

REQUIRED FILES:
1. favicon.ico - Multi-resolution ICO file (16x16, 32x32, 48x48)
2. logo192.png - 192x192px PNG for PWA
3. logo512.png - 512x512px PNG for PWA

DESIGN RECOMMENDATIONS:
----------------------
The admin favicon should be DISTINCT from the user-facing app to help
administrators quickly identify the admin panel in their browser tabs.

Suggested Design Elements:
- Color Scheme: Dark/professional colors (navy, charcoal, forest green)
- Icon Type: Dashboard, chart, lock, shield, or gear icon
- Badge: Consider adding a small "A" or "Admin" badge
- Contrast: Ensure good visibility on both light and dark browser tabs

IMPLEMENTATION STEPS:
--------------------
1. Design or select your admin icon (512x512px PNG)
2. Use favicon.io or RealFaviconGenerator to create:
   - favicon.ico
   - logo192.png
   - logo512.png
3. Replace this file and the placeholder PNG files
4. Test in browser (clear cache to see changes)

EXAMPLE DESIGN IDEAS:
--------------------
1. Dashboard Icon: Grid of squares representing dashboard widgets
2. Admin Badge: Your main logo with "ADMIN" badge overlay
3. Chart Icon: Bar chart or pie chart in professional colors
4. Shield Icon: Security/admin access indicator
5. Control Panel: Sliders/knobs indicating control interface

COLOR SUGGESTIONS:
-----------------
- Primary: #1a202c (Dark Charcoal) - Matches theme
- Accent: #4299E1 (Professional Blue)
- Alternative: #48BB78 (Success Green)
- Professional: #5A67D8 (Indigo)

TESTING:
--------
After adding your favicon:
1. Clear browser cache
2. Open admin panel in browser
3. Check favicon appears in:
   - Browser tab
   - Bookmarks
   - PWA install dialog
   - Mobile home screen (if installed as PWA)
